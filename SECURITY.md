# Güvenlik Politikası

## Güvenlik Açığı Bildirme

Bir güvenlik açığı keşfettiysen lütfen bunu doğrudan issue olarak açma.

Bunun yerine e-posta ile bildir: **mehmet@garage.ist**

Bildiriminde şunları belirt:
- Açığın türü
- Etkilenen dosya veya alan
- Nasıl tetiklenebileceği
- Varsa olası etkisi

## Yanıt Süreci

- 48 saat içinde dönüş yapılır
- Geçerli açıklar gizli tutulur, düzeltildikten sonra açıklanabilir
